# Bakasura — Feed The Beast

Bakasura is a premium South Indian culinary experience that celebrates authentic regional flavors, rustic traditions, and the art of storytelling through food. From the buttery Benne Dosas of Bengaluru to the fiery Hunter's Lamb Roast, every dish is crafted with honesty and memory.

## 🏺 Experience the Story
The Bakasura website is a high-fidelity digital gateway to our culinary world, featuring:
- **Art of Flavors**: A cinematic journey through our signature regional dishes.
- **Weekend Feasts**: Grand celebrations of community and cuisine.
- **Chef's Philosophy**: Insights into our commitment to authentic, soul-touching food.

## 🛠️ Technology Stack
- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Styling**: Vanilla CSS & Tailwind CSS
- **Typography**: Custom Mulish & Serif font rhythm
- **Optimization**: Next/Image & Advanced Parallax effects

## 🚀 Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/NOTREGRETO/Bakasura.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

### Deployment
The project is optimized for deployment on **Vercel**. Connect your GitHub repository to Vercel for automatic CI/CD.
